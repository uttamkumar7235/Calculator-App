## 📋 How It Works

1. The user clicks calculator buttons or presses keyboard keys.
2. The entered values are stored as a mathematical expression.
3. When the **=** button (or **Enter** key) is pressed, the expression is evaluated.
4. The result is displayed in the calculator screen.
5. Invalid expressions are handled gracefully and display an error message.
